# Spotify Music Downloader
## Description
```
A python app which allows you to keep a local copy of your favourite spotify tracks while enjoying them.
```
## Getting Started

### Dependencies
```
* YoutubeSearchPython
* Requests
* Pafy
* Tkinter
* Spotify API 
* Python Version 3
```
## Help
```
Advised version of python to be used : 3
```
